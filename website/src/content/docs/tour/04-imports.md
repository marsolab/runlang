The word `use` is used to import packages into your program.

```go
use "fmt"
```

## Import paths

Import paths are the path to the package. They are used to import packages into your program.

```go
use "math/rand"
```

It can also be written as a grouped import:

```go
use (
    "math/rand"
    "fmt"
)
```
